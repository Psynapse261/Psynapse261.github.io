Server Platform Debug Intern at AMD, Austin, Texas

AMD’s EPYC server processors power some of the most advanced data centers, delivering high-performance computing with innovative architectures. As part of my internship at AMD, I investigate and resolve hardware and software issues on EPYC server platforms, addressing challenges related to SoC-level IPs, DRAM stability, and IO devices using protocols like USB, PCIe, and CXL.
I analyze system failures involving drivers, BIOS, and operating systems, collaborating with cross-functional teams to perform root cause analysis and implement effective solutions. To streamline issue detection and system validation, I develop Python and Bash scripts for triage and test automation, significantly improving debugging efficiency. My work contributes to the overall reliability and performance optimization of AMD’s EPYC platforms, which are at the forefront of modern server technology.
Key Takeaways from the experience:

  • Server Platform Triage and Debug
  • Firmware and OS Level Analysis
  • PCIe, CXL, DRAM, USB protocols

Software Developer Intern at GMRT, Pune, India

The Giant Metrewave Radio Telescope is a radio astronomy telescope having an array of 30 antennas each 40 metres wide, and is used for observations of pulsars, quasars, etc in the radio bands.
I worked with their software backend system to integrate a time series signal processing system in the pipeline. This was a filtering algorithm which had to run in real-time on raw digital data received from the antennas. I first utilized multithreading techniques to parallelize time series signal processing algorithms, significantly improving computational efficiency and reducing processing time by 10%. Then I developed and optimized CUDA kernels for GPU acceleration, harnessing the parallel processing power of NVIDIA GPUs to achieve remarkable speedup of 100% in time series data filtering.
Key Takeaways from the experience:

  • High Performance Computing
  • Real-Time Integration
  • Efficient Hardware Utilization

Embedded Systems Intern at Makers Lab, Tech Mahindra, Pune, India

The Makers Lab at Tech Mahindra India is an innovative research and development unit which is focused on connecting with academia, businesses, and governments across the world to bring about a positive change in their lives by using cutting edge technologies.
I worked at the lab on agritech module designed to monitor environmental conditions of the farm remotely. I implemented sensor interfaces for monitoring soil moisture, temperature, and humidity, enhancing precision agriculture practices and optimizing irrigation strategies. I implemented power-efficient algorithms and low-power modes to extend the battery life of embedded sensor nodes, enabling prolonged field deployment without frequent maintenance. I also developed communication pipelines utilizing wireless protocols such as Zigbee and HTTP over GSM to transmit sensor data to a central server, ensuring seamless connectivity across the agricultural field.
Key Takeaways from the experience:

  • Product Development Cycles
  • Power Efficient Software Development
  • Remote Sensing Technologies
