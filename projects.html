Processor Enhancements using Gem5

As part of a course project, I analyzed the O3 processor's performance using Gem5, exploring predictor size, pipeline width, execution units, and branch predictors. The goal of these projects was to gain a practical understanding of a processor's internal workings and performance/efficiency enhancements through developing and modifying different elements in modern day CPUs. I examined a customized GShare Branch Predictor in C++, evaluating predictor size, counter bits, and history bits. Additionally, I implemented the Insertion Promotion Vector-based LRU Cache Replacement Policy and experimented with L2 cache sizes to optimize memory hierarchy performance.

  • Branch Prediction
  • Cache Replacement Policies
  • Performance Analysis with Gem5
